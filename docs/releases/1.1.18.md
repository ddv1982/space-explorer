# Release 1.1.18 — moon-surface prop cleanup

## Highlights

- Removes the remaining generated moon-surface prop overlays that were still appearing over premium backgrounds on the affected levels.
- Keeps the simpler moon-surface terrain/horizon/crater treatment while deleting the distracting prop-style geometry.
- Simplifies the related config surface and adds focused regression coverage around the generator.

## What changed

- Moon-surface prop removal:
  - `src/systems/parallax/moonSurfaceGenerator.ts`
  - Removed generated building bodies, windows, antenna lights, accent spires, and runway/light-strip props.

- Moon-surface config cleanup:
  - `src/config/levels/types.ts`
  - `src/config/levels/definitions/ionStormSector.ts`
  - `src/config/levels/definitions/nebulaPass.ts`
  - `src/config/levels/definitions/wreckfieldRun.ts`
  - Removed the now-unused `buildingCount` field from the moon-surface config shape and the three affected level configs.

- Regression coverage updates:
  - `tests/moonSurfaceGenerator.test.ts`
  - `tests/moonSurfaceLayerLifecycle.test.ts`
  - Added focused coverage proving the generator no longer emits prop-style geometry while preserving lifecycle behavior.
