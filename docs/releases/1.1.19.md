# Release 1.1.19 — premium background presentation cleanup

## Highlights

- Stops rendering the old procedural starfield underneath premium background art.
- Makes premium background layers fully opaque so the level base background color no longer bleeds through the dark center lane.
- Adds focused regression coverage around the premium-background path.

## What changed

- Premium background presentation cleanup:
  - `src/systems/ParallaxBackground.ts`
  - `src/systems/parallax/premiumBackgroundManifest.ts`
  - Premium-art levels now skip the legacy procedural starfield pass, and premium background alpha is now fully opaque.

- Regression coverage updates:
  - `tests/ParallaxBackground.test.ts`
  - Added coverage proving premium background levels skip starfield creation when premium art is available.
