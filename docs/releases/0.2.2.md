# Space Explorer 0.2.2

Release date: 2026-04-07

## Highlights

- Performance now prioritizes smoothness with conservative runtime fallback logic for sustained low-FPS situations.
- Level 1 (Asteroid Belt Alpha) now keeps its dark mood while presenting more vibrant cyan color energy.

## Performance and hardware load

- Added conditional FPS logic as a **safety net**:
  - 5-second grace period after gameplay start
  - two consecutive low-FPS sample windows required before fallback triggers
- When triggered, optional visual cost is reduced (global glow, burst density, and background motion/detail load).

## Visual tuning

- Retuned Level 1 palette and grade to increase vibrancy while preserving dark-scene readability and upgrade feel.
