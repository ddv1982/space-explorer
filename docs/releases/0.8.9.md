# Release 0.8.9 — Helper-wing transition crash fix

## Highlights

- This patch fixes a runtime crash that could happen when finishing a level while helper-wing support was active.
- Focus: robust helper lifecycle teardown/suspension across Game → intermission transitions.

## What changed

- Runtime lifecycle hardening:
  - `src/systems/LastLifeHelperWing.ts`
  - Added explicit overlap-collider tracking and destruction during helper-wing cleanup.
  - Added `suspendForTransition()` to deactivate active helpers before scene handoff.

- Transition hook integration:
  - `src/scenes/GameScene.ts`
  - Level-complete and boss-death completion paths now suspend helper wing before queueing terminal transition flow.

## Bug fixed

- Fixed crash path observed after late-level completion with helper wing active:
  - `TypeError: undefined is not an object (evaluating 'n.forEach')`

- `bun run levels:validate`
- `bun test`
