# Release 1.1.5 — Runtime refactor and export cleanup

## Highlights

- Keeps the `1.1.x` gameplay line intact while applying a broader internal cleanup across runtime orchestration, collision handling, pooling, and module exports.
- Breaks up `GameScene` into clearer private phases, reducing orchestration complexity without changing scene ownership.
- Consolidates duplicate player-damage routing and pool setup logic behind tested internal helpers.
- Prunes unused exports and local-only type surfaces so the repo’s actual API matches what the codebase really uses.
- Brings `knip` fully green while keeping tests, build, and lint green.

## What changed

- Runtime refactor seams:
  - `src/scenes/GameScene.ts`
