# Release 0.5.4 — Boss HP escalation pass

## Highlights

- Bosses were still too easy to burn down, especially in upgrade-heavy runs.
- This release applies a direct durability escalation with a much steeper four-digit boss HP ladder.

## Gameplay tuning

- Updated boss base HP values:
  - Arc Warden Khepri: `54 -> 1200`
  - Marshal Bront: `70 -> 1700`
  - Pyre Leviathan: `84 -> 2300`
  - Reliquary Crown: `90 -> 2900`
  - Archivist Shade: `96 -> 3600`
  - Orbital Cantor: `104 -> 4400`
  - Rift Basilisk: `112 -> 5300`
  - Axiom Null: `126 -> 6400`

This preserves strict campaign escalation while making every boss encounter significantly longer and more demanding.
