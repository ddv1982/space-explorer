# Release 1.1.4 — Refactor quality patch

## Highlights

- Keeps the `1.1.x` gameplay line intact while packaging a small internal quality pass as a release-ready patch.
- Simplifies pause-state synchronization so pause presentation and mobile-control blocking follow one shared update path.
- Consolidates low-risk level-config helper duplication without changing authored level content or music output.
- Locks the cleanup behind new regression tests for pause semantics, campaign validation, and arrangement immutability.
