# Space Explorer 0.2.0

Release date: 2026-04-07

## Highlights

- Upgraded engine baseline to **Phaser 4.0.0-rc.7**.
- Refreshed procedural ship models with stronger silhouettes, richer paneling, and improved thruster heat/bloom detail.
- Improved scene readability and atmosphere with denser star layers, upgraded nebula rendering, softer particles, and refined camera filters.

## Technical notes

- Replaced Phaser 3 `preFX` / `postFX` calls with Phaser 4 FilterList APIs (`camera.filters`, `gameObject.enableFilters()`).
- Kept the project's procedural-art architecture; no external heavy model assets were introduced.

## Validation

- `bun run build`
- `bun run lint`
