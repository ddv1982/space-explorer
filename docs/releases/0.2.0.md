# Space Explorer 0.2.0

Release date: 2026-04-07

## Highlights

- Upgraded runtime engine from Phaser 3.90 to **Phaser 4.0.0-rc.7**.
- Reworked rendering integration to align with Phaser 4 filter architecture.
- Applied a compatibility-first migration strategy to keep gameplay baseline stable before visual expansion.

## Architecture changes

- Added `src/utils/renderingCompat.ts` to centralize renderer/filter integration and reduce duplicated logic.
- Migrated game object glow handling from Phaser 3 `preFX` to Phaser 4 filters in:
  - `src/entities/Player.ts`
  - `src/entities/PowerUp.ts`
- Migrated camera FX and color grading in `src/systems/EffectsManager.ts` from `postFX` assumptions to Phaser 4 filter lists.

## Validation

- `bun run lint`
- `bun run knip`
- `bun run build`

## Notes

- This release is intentionally compatibility-first. Further visual/performance enhancements will be layered after the migration baseline proves stable in playtesting.
