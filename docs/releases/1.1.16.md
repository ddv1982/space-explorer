# Release 1.1.16 — gameScene maintainability tranche

## Highlights

- Splits the dense private `gameScene` bootstrap contract into phase-local files so runtime, world, input, gameplay, HUD, and pause ownership are easier to navigate.
- Finishes the remaining cross-feature import cleanup inside `src/scenes/gameScene/` using the existing `@/*` alias.
- Preserves the explicit create-time orchestration flow and existing gameplay behavior.
