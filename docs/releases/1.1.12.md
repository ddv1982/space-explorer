# Release 1.1.12 — Mobile joystick control tuning

## Highlights

- Makes the mobile joystick feel calmer and more deliberate near center so small thumb drift no longer triggers movement as eagerly.
- Stabilizes held directions and reduces accidental weak diagonals so mobile movement feels closer to intentional desktop inputs.

## What changed

- Mobile joystick deadzone tuning:
  - `src/systems/MobileControls.ts`
  - Added a small deadzone and remapped usable stick throw so movement starts farther from the joystick center instead of reacting immediately to tiny touch offsets.

- Mobile direction gating:
  - `src/systems/InputManager.ts`
  - Replaced the simple one-shot mobile threshold with engage/release hysteresis, stronger reverse-direction commitment, weak-secondary-axis suppression, and cached direction resolution so repeated reads stay consistent.

- Regression coverage:
  - `tests/InputManager.test.ts`
  - Added tests that lock deliberate direction engagement, held-direction hysteresis, reverse-direction gating, and weak-diagonal suppression behavior.
