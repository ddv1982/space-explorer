# Release 0.8.16 — Follow-up guardrails and scene-flow confidence

## Highlights

- Adds explicit bundle monitoring commands to catch size regressions early without suppressing existing Vite warning signals.
- Improves scene transition confidence with additional edge-case tests and a lightweight manual smoke checklist.
- Keeps gameplay behavior unchanged while strengthening release safety checks.

## What changed

- Bundle guardrails:
  - `scripts/bundle-report.mjs`
  - `package.json`
  - `docs/bundle-size-optimization-notes.md`
  - Added:
    - `bun run bundle:report` (reports raw+gzip output totals and largest files)
    - `bun run bundle:check` (fails on configurable threshold regressions)

- Scene-flow confidence:
  - `tests/sceneRegistry.test.ts`
  - `docs/qa/scene-flow-smoke-checklist.md`
  - Added focused transition safety coverage for:
    - already-registered scene skip path,
    - loader-failure no-start behavior.
  - Added a concise manual flow checklist for Menu/Game/Intermission/GameOver/Victory transitions.

## Validation

- `bun run levels:validate`
- `bun test`
- `bun run build`
- `bun run bundle:check`
