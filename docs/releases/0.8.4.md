# Release 0.8.4 — Phase 2 campaign pacing uplift

## Highlights

- This release extends the v0.8.3 pacing/readability framework to the remaining campaign levels.
- Goal: preserve high difficulty while making escalation and telegraph timing more legible and enjoyable.

## What changed

- Retuned core tail levels:
  - `src/config/levels/definitions/warzoneCorridor.ts`
  - `src/config/levels/definitions/deepSpaceInferno.ts`
- Retuned expansion arc levels:
  - `src/config/levels/definitions/wreckfieldRun.ts`
  - `src/config/levels/definitions/ghostlightVeil.ts`
  - `src/config/levels/definitions/crownOfRings.ts`
  - `src/config/levels/definitions/obsidianMaw.ts`
  - `src/config/levels/definitions/terminusBlack.ts`
- Across these levels:
  - hazard cadence/intensity values were smoothed to reduce unreadable spike transitions,
  - spawn pressure and climax carryover were rebalanced,
  - explicit section-level `tensionArc` and `vatTarget` fields were added for clearer emotional contouring,
  - boss-approach sections were tuned to improve anticipation clarity before boss encounters.
