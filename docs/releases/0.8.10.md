# Release 0.8.10 — Persistent helper wing across levels

## Highlights

- Helper-wing support now persists across level transitions during a run.
- If you keep surviving on last life, helper support can grow into both side slots over time.
- Transition robustness from 0.8.9 is retained.

## What changed

- Persistent helper state in player run data:
  - `src/systems/PlayerState.ts`
  - Added `helperWing.slots` state for helper remaining lives + HP.

- Helper entity state serialization/restore helpers:
  - `src/entities/HelperShip.ts`
  - Added helper persistent-state read/apply methods.

- Runtime persistence and slot fill behavior:
  - `src/systems/LastLifeHelperWing.ts`
  - Restores surviving helpers at level start.
  - Preserves transition-safety cleanup.
  - Allows new helper grants to fill empty support slots when available.

- Scene integration:
  - `src/scenes/GameScene.ts`
  - Persists helper state on level completion / boss completion and scene shutdown.
  - Restores helper state when entering the next level.

## Gameplay result

- A helper you earned no longer disappears just because the level changed.
- Surviving helper ships remain with their own lives/HP until depleted.
- You can build back toward two active support helpers across later levels.
