# Release 0.7.3 — Music simplification and consistency pass

## Highlights

- This patch simplifies the procedural music behavior to improve cross-level consistency and reduce unnecessary complexity.
- Focus areas are global defaults, Nebula override cleanup, and campaign-wide complexity guardrails.

## What changed

- Normalized shared music defaults in `reimaginedMusic.ts`:
  - reduced default phase scatter,
  - reduced accent volatility,
  - simplified accent patterns,
  - tightened arrangement density/energy/layer-gain ranges for steadier section continuity.
- Simplified Nebula Pass (`Prism Reef`) rhythm overrides in `nebulaPass.ts`:
  - flattened phase scatter,
  - reduced accent complexity,
  - retained 7/8 identity with 7/14-aligned divisions,
  - tuned gates for more stable cadence.
- Added campaign-wide complexity guardrail tests in `tests/musicComplexityGuardrails.test.ts`:
  - rhythm complexity budget checks,
  - odd-meter alignment checks,
  - arrangement arc ordering checks,
  - deterministic aggregated diagnostics for regression clarity.
