# Release 0.7.2 — Level 2 music stability pass

## Highlights

- This patch targets the reported level-to-level inconsistency where level 1 felt stable but level 2 felt irregular.
- Focus is a data-first stabilization for Nebula Pass plus safety guardrails and deterministic regression coverage.

## What changed

- Simplified Level 2 (`Prism Reef`) rhythm overrides in `nebulaPass.ts`:
  - kept 7/8 identity,
  - aligned subdivisions toward 7/14,
  - raised gate floor,
  - reduced phase/accent volatility.
- Added warning-only early-level rhythm guardrails in `scripts/validateLevels.ts`:
  - odd-meter division alignment diagnostics,
  - low-gate warnings,
  - phase-scatter warnings,
  - beatUnit-aware bar math.
- Added deterministic Nebula-style 7/8 cadence regression assertions in `tests/polyrhythm.test.ts`:
  - multi-bar trigger step maps,
  - aggregate trigger counts,
  - explicit stability lock for future refactors.
