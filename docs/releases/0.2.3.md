# Space Explorer 0.2.3

Release date: 2026-04-07

## Highlights

- Reworked performance tuning using research-backed Phaser practices and removed duplicated runtime policy logic.
- Restored a darker mood in Level 1 while keeping a vibrant cyan-forward visual upgrade feel.

## Performance updates

- Introduced a shared performance policy module to centralize device-tier and sustained-FPS fallback behavior.
- Added particle emitter lifecycle bounds and camera culling hints:
  - `maxAliveParticles`
  - `maxParticles`
  - `viewBounds`
- Reduced runtime config churn by caching keyed emitter configurations (explosions and power-up burst tinting).

## Visual tuning

- Rebalanced Asteroid Belt Alpha palette/grade to be darker overall with controlled high-energy accent color.
