# Release 1.1.7 — Visual framing cleanup

## Highlights

- Removes the shared Phaser-drawn edge framing shapes that were reading as cone-like overlays in multiple levels.
- Keeps authored level pacing and hazard logic intact while simplifying presentation around the playfield edges.
- Removes both the foreground silhouette layer and the `rock-corridor` overlay visuals, leaving a cleaner play space.

## What changed

- Foreground silhouette removal:
  - `src/systems/parallax/foregroundSilhouetteLifecycle.ts`
  - `tests/foregroundSilhouetteLifecycle.test.ts`
  - Disabled the shared side-silhouette generator entirely and updated lifecycle coverage to lock the no-op behavior.

- Rock corridor overlay removal:
  - `src/systems/parallax/hazardOverlayRenderer.ts`
  - `tests/hazardOverlayRenderer.test.ts`
  - Removed the shared `rock-corridor` overlay primitives while preserving the rest of the hazard overlay renderer behavior.
