# Release 1.1.0 — Visual overhaul

## Highlights

- Ships now read more distinctly at gameplay scale through stronger procedural silhouette language.
- Levels react more clearly to pacing with section-driven atmosphere and hazard-linked scenic response.
- Bosses now present stronger identity through shared attack-style-based visual variants.
- Finale spectacle is concentrated in `Eventide Singularity` without abandoning the project’s readability-first rules.

## Included in 1.1.0

- Ship and projectile presentation:
  - stronger player/helper/enemy silhouettes,
  - active player and enemy bullet trails,
  - retained shared procedural sprite pipeline.

- Environmental presentation:
  - section-driven atmosphere modulation,
  - compact per-section `visualModifiers` authoring support,
  - hazard-linked scenic overlays,
  - stronger landmark identity for `Prism Reef` and `Magnetar Foundry`,
  - edge-only foreground silhouette depth treatment.

- Boss and spectacle presentation:
  - attack-style-based boss visual variants,
  - reusable transient camera color pulses,
  - stronger showcase set-piece treatment for `Eventide Singularity`.
