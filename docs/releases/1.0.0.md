# Release 1.0.0 — Major baseline reset

## Highlights

- Establishes `v1.0.0` as the new primary release baseline.
- Resets historical remote release/tag stream and starts a clean major-version track.
- Consolidates recent runtime smoothness and bundle/startup improvements under one stable major release.

## Included in 1.0.0 baseline

- Runtime performance/smoothness:
  - pooled physics lifecycle hardening,
  - resize/parallax churn reductions,
  - death/respawn hitch smoothing and transition polish.

- Bundle and startup optimization:
  - lazy scene registration for heavy scenes,
  - explicit chunking strategy for lean entry payload,
  - bundle guardrail tooling via:
    - `bun run bundle:report`
    - `bun run bundle:check`

- Confidence and QA:
  - scene registry edge-case tests,
  - release smoke checklist for critical scene transitions.

## Validation

- `bun run levels:validate`
- `bun test`
- `bun run build`
- `bun run bundle:check`
