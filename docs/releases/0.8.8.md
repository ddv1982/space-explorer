# Release 0.8.8 — Last-life helper wing assist

## Highlights

- Added a late-game survival assist for selected boss-heavy levels: a helper wing that activates on your last life.
- Helpers follow your ship, fire support shots, and remain active until their helper lives are fully spent.

## What changed

- Runtime helper-wing system:
  - `src/entities/HelperShip.ts`
  - `src/systems/LastLifeHelperWing.ts`
  - New pooled helper ships with follow steering, support fire, damage handling, respawn, and depletion lifecycle.

- Scene + HUD integration:
  - `src/scenes/GameScene.ts`
  - `src/systems/GameplayFlow.ts`
  - `src/systems/HUD.ts`
  - Added helper-wing activation/depletion gameplay events and announcement feedback in HUD.

- Config extension:
  - `src/config/levels/types.ts`
  - Added optional `lastLifeHelperWing` level tuning block to gate/author helper assist per level.

- Helper visuals:
  - `src/utils/SpriteFactory.ts`
  - Added generated helper-ship texture.

- Late-level rollout (opt-in):
  - `src/config/levels/definitions/wreckfieldRun.ts`

- Research note:

## Gameplay behavior

- On configured late levels, when the run reaches **remaining lives = 1**, the helper wing activates.
- Default authored behavior in this release:
  - 2 helper ships
  - each with ~50% of player max HP
  - each with 2 helper lives before permanent loss
  - helper shots use the existing allied bullet pipeline for consistency.
